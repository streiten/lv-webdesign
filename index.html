Examples for triggering and controlling videos, p5, lottie and svgs using anime.js and scrollmagic.

video

trigger and pause playback of a html5 video. rotation is bound to scrollposition by updating a css variable.
(make sure to use video withouht sound or mute. otherwise starting playback will not work.)

p5js

pass variables to a p5 sketch.
use instance mode for embedding p5 in websites.

lottie

stop an autoplaying, looping lottie animation and advance playback on scrolling (either direction).

anime

just an anime.js svg animation. no scrollmagic integration.

svg line drawing

animate an svg path bound to scrollposition using anime.

parallax background

a classic example of animating a css property using scrollposition.

just space for allowing to leave the previous section.
